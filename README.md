A library to convert currency using [Google Finance](https://www.google.com/finance).

---

###Installation

For Python 2 users:

`sudo pip install Currency-Converter`

For Python3 users:

`sudo pip3 install PyCurrency`

Alternatively you can install git clone this repository and install it via `setup.py`.

`git clone https://github.com/SiddharthSaxena/PyCurrency.git`

`python setup.py install` or `python3 setup.py install`
